# Publication Studio front-end

### Requirements
* [brew](https://brew.sh/)
* nvm (to install node.js)
* node.js 6.10+
* yarn

### Machine Setup

```
brew install nvm yarn
nvm install 6
nvm alias default 6
```

### Repository Setup

```
yarn install
yarn start
```

### Connect and enjoy!

[http://publication.studio.local:5001](http://publication.studio.local:5001)
